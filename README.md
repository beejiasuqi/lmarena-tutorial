# LMArena 使用教程

一个介绍如何免费使用顶级AI模型平台LMArena的教程网站。

## 在线访问

访问 [GitHub Pages](https://你的用户名.github.io/仓库名/) 查看完整教程。

## 关于 LMArena

LMArena 是一个创新性的AI模型评估平台，您可以在这里免费体验：
- GPT-5.2、Gemini 3 Pro、Claude Opus 4.5 等顶级文本模型
- Nano Banana Pro、FLUX 2 等图像生成模型
- Veo 3.1、Sora 2 等视频生成模型

## 技术栈

- React 19
- Tailwind CSS 4
- Vite
- TypeScript

## 本地开发

如果您想在本地运行此项目：

```bash
# 安装依赖（需要Node.js 18+）
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
